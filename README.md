# QuestAnsApp
Final Capstone

A frontend for the feathers-express app using React.

To run, follow the following instructions:

- cd to top of QuestAnsApp directory 
- Run 'npm install'
- cd to feathers-question-answer-chat directory
- Run 'npm install'

Now 
- Make sure the feathers-question-answer-chat API is running on localhost port 3030 by running 'npm start' while still on the directory
- Move back over to QuestAnsApp directory and do 'run start' - this will be running on localhost port 3000
